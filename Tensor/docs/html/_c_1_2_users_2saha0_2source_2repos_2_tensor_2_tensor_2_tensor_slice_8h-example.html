Const indexing that returns an independent Tensor copy.

Const indexing that returns an independent Tensor copy.Extends the index chain and immediately returns an independent copy. Used for read-only access.

Parameters
idxThe next index in the chain.
Returns
A new independent Tensor at the indexed location.

const Tensor& A = ...; auto slice = A[0][1]; // Returns independent copy
